Introduction

Python Example


def earnings_growth(revenue, expenses):
    profit = revenue - expenses
    growth_rate = profit / revenue
    return growth_rate

Mathematical Model

We define profit as:

\\( \text{Profit} = \text{Revenue} - \text{Expenses} \\)
