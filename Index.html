Key Financial Metrics

Return on Equity (ROE)

$$ROE = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} = \frac{340M}{2.24B} = 15.2\%$$

Net Interest Margin (NIM)

$$NIM = \frac{\text{Interest Income} - \text{Interest Expense}}{\text{Average Earning Assets}} = 3.45\%$$

Efficiency Ratio

$$\text{Efficiency Ratio} = \frac{\text{Non-Interest Expenses}}{\text{Net Revenue}} = 42\%$$

These formulas show our strong profitability metrics. ROE of 15.2% exceeds our target and peer averages. NIM improvement reflects better asset-liability management.

# Credit Risk Score Calculation
import numpy as np
import pandas as pd
from sklearn.ensemble import RandomForestClassifier

class CreditRiskModel:
    def __init__(self):
        self.model = RandomForestClassifier(n_estimators=100)
        self.risk_weights = {
            'payment_history': 0.35,
            'credit_utilization': 0.30,
            'account_age': 0.15,
            'credit_mix': 0.10,
            'new_credit': 0.10
        }
    
    def calculate_risk_score(self, customer_data):
        """Calculate composite credit risk score"""
        features = self._extract_features(customer_data)
        probability = self.model.predict_proba(features)[0][1]
        
        # Convert to risk score (300-850 scale)
        risk_score = 300 + (probability * 550)
        return min(max(risk_score, 300), 850)
    
    def portfolio_risk_assessment(self, loan_portfolio):
        """Assess overall portfolio risk"""
        total_exposure = loan_portfolio['loan_amount'].sum()
        weighted_risk = np.average(
            loan_portfolio['risk_score'], 
            weights=loan_portfolio['loan_amount']
        )
        return {
            'total_exposure': total_exposure,
            'weighted_avg_risk': weighted_risk,
            'var_95': self._calculate_var(loan_portfolio, 0.95)
        }
